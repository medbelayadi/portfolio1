SmartCima

SmartCima is a sleek movie ticket booking app with a modern purple gradient UI and intuitive bottom navigation. It features categorized movie listings (Now Playing, Coming Soon, Top Movies) with scrollable posters and detailed views. Users can select showtimes, pick seats on an interactive chart, and securely complete payments. The app generates digital tickets with barcodes and stores user info in a personalized profile. Additional features include search, favorite movies, saved payment methods, and notification preferences..
